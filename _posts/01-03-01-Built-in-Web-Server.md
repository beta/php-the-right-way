## 内置的 web 服务器 {#builtin_web_server_title}

从 PHP 5.4 开始，你可以在不需要安装和配置一个功能完整的 web 服务器的情况下学习 PHP。要启动这个内置的服务器，在终端中进入项目的 web 根目录然后执行以下命令：

{% highlight console %}
> php -S localhost:8000
{% endhighlight %}
